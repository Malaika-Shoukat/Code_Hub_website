Variables & Datatypes

Learn the basic building blocks of programming: variables and data types.

What is a Variable?

A variable is a container used to store data in a program. You can think of it as a labeled box that holds a value.

Definition: A variable is a storage location in memory with a name and a value.

Example of Variables

name = "Ali"
age = 20
height = 5.9

Rules for Naming Variables

  • Must start with a letter or underscore (_)
  • Cannot start with a number
  • No spaces allowed
  • Use meaningful names (recommended)
  • Case-sensitive (Age ≠ age)

What are Datatypes?

Datatype refers to the kind of data a variable can store. Programming languages support several built-in datatypes.

Tip: Choosing the right data type makes your program efficient and organized.

Common Datatypes

Datatype Description Example
Integer (int) Whole numbers without decimals age = 18
Float / Double Numbers with decimals height = 5.7
String Text or characters name = "Saliha"
Boolean True/False values isStudent = true

Example Program

-- Variables in a program
name = "Sana"
quantity = 5
price = 120.5

-- Calculate total
total = quantity * price
print(total)

Why Variables & Datatypes Matter?

  • Store and manage information
  • Perform calculations and logic
  • Control program flow
  • Essential for every programming language

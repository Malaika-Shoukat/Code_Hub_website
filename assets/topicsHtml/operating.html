Chapter 6: Operating System

Understanding how computers work internally.

What is an Operating System?

An Operating System (OS) is system software that manages hardware, software applications, memory, and user interaction. It acts as a bridge between the user and the computer hardware.

Definition:
An Operating System is software that controls the overall working of a computer and manages all hardware and software resources.

Main Functions of an Operating System

  • Process Management: Handles execution and scheduling of programs.
  • Memory Management: Allocates RAM to processes.
  • File Management: Organizes files and folders.
  • Device Management: Controls input/output devices.
  • Security: User authentication, password protection, permissions.
  • User Interface: GUI (Graphical) or CLI (Command line).

Types of Operating Systems

  • Batch Operating System
  • Time-Sharing / Multitasking OS
  • Distributed Operating System
  • Real-Time Operating System
  • Mobile OS (Android, iOS)

Examples of Operating Systems

Operating System Type
Windows Desktop OS
macOS Desktop OS
Linux / Ubuntu Open-source OS
Android Mobile OS
iOS Mobile OS

Process States in an Operating System

Every program (process) goes through these states:

New → Ready → Running → Waiting → Terminated

Memory Management Techniques

  • Paging: Memory divided into equal-size pages.
  • Segmentation: Divided logically into segments.
  • Virtual Memory: Hard disk used as temporary RAM.
Why Operating Systems Matter?
Without an OS, users cannot interact with the computer, and hardware cannot function properly.

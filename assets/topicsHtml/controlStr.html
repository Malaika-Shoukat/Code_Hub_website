Control Structures

Understand how decisions and repetitions are handled in programming.

What Are Control Structures?

Control Structures define the flow of a program — how decisions are made and how actions are repeated.

Definition: Control structures allow a program to execute instructions conditionally or repeatedly.

Types of Control Structures

  1. Selection / Decision Making (if, else-if, else)
  2. Looping / Repetition (for, while, do-while)
  3. Jump Statements (break, continue)

1. Selection Statements

IF Statement

Executes a block of code only if a condition is true.

if (age > 18) {
   print("You are an adult");
}

IF-ELSE Statement

if (marks >= 50) {
   print("Pass");
} else {
   print("Fail");
}

ELSE IF Ladder

if (grade >= 90) {
   print("A");
} else if (grade >= 80) {
   print("B");
} else {
   print("C");
}

2. Looping Statements

FOR Loop

Used when you know how many times to repeat.

for (i = 1; i <= 5; i++) {
   print(i);
}

WHILE Loop

Runs as long as the condition is true.

while (count <= 3) {
   print("Hello");
   count++;
}

DO-WHILE Loop

Runs at least once before checking the condition.

do {
   print("Welcome");
   x++;
} while (x < 5);

3. Jump Statements

BREAK

Stops the loop immediately.

for (i = 1; i <= 10; i++) {
   if (i == 5) break;
}

CONTINUE

Skips the current iteration.

for (i = 1; i <= 5; i++) {
   if (i == 3) continue;
   print(i);
}
Tip: Mastering control structures is essential because all logic in programming depends on conditions and loops.

Chapter 5: Databases and SQL

Learn how data is stored, managed, and accessed using databases and SQL.

What is a Database?

A Database is an organized collection of data that can be easily accessed, managed, and updated. It stores information in a structured format, typically in tables made up of rows and columns.

Definition: A database is a structured system for storing, retrieving, and managing data efficiently.

Why Use Databases?

  • Store large amounts of data securely.
  • Reduce duplication and ensure consistency.
  • Quickly search, sort, and filter data.
  • Support multiple users accessing the same data simultaneously.

Components of a Database

  • Table: Collection of related data organized in rows and columns.
  • Record (Row): Represents one data item (e.g., one student’s details).
  • Field (Column): A single attribute (e.g., Name, Age, Grade).
  • Primary Key: A unique identifier for each record in a table.

What is SQL?

SQL (Structured Query Language) is a standard programming language used to manage and manipulate databases. It allows users to create, read, update, and delete data in a relational database.

Fun Fact: SQL is pronounced as “Sequel” and was first developed by IBM in the 1970s.

Common SQL Commands

Command Purpose Example
CREATE TABLE Creates a new table in the database. CREATE TABLE Students (ID INT, Name TEXT);
INSERT INTO Adds new data into a table. INSERT INTO Students VALUES (101, 'Ali');
SELECT Retrieves data from one or more tables. SELECT * FROM Students;
UPDATE Modifies existing records. UPDATE Students SET Grade='A' WHERE ID=101;
DELETE Deletes records from a table. DELETE FROM Students WHERE ID=102;

Example SQL Program

-- Create a table
CREATE TABLE Students (
  ID INT PRIMARY KEY,
  Name TEXT,
  Grade TEXT
);

-- Insert records
INSERT INTO Students VALUES (101, 'Ali', 'A');
INSERT INTO Students VALUES (102, 'Sana', 'B');

-- View records
SELECT * FROM Students;

-- Update a grade
UPDATE Students SET Grade = 'A+' WHERE ID = 102;

Benefits of Using SQL Databases

  • Data consistency and integrity.
  • Security and controlled access.
  • Support for large-scale data handling.
  • Widely used across industries and web applications.
Tip: SQL is a must-learn skill for anyone interested in data science, AI, or software development!
